# Fly-in

Description

A drone routing simulation that parses map files, computes paths, and simulates
multiple drones moving from a start hub to an end hub under capacity and
movement constraints.

Instructions

- Install dependencies: see `requirements.txt` and run `make install`.
- Run the default example: `make run` (uses `maps/test/00_TEST.txt`).

Resources

- Pydantic for parsing helpers
- AI was used to audit the repository and add tooling files.
